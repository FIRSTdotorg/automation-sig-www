# Amazon S3

Scalable, cloud object storage by Amazon


**Open Source:** No

**Category:** Storage


**Core Features:**
- Scalable
- Supported storage backend by a lot of tooling
- Supports encrypting your data at rest and in transit
- Open source tools exist that implement its API
